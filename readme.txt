== Description ==

This Plugin add an filetype related icon to all your linked files in content, excerpts, comments and widgets. 
With this icons all your visitors will know what kind of file they will get.

*If this plugin has helped you, please take a moment to [rate
it](http://wordpress.org/support/view/plugin-reviews/filetype-icons#postform)!*

== Installation ==

1. Upload all files and folders to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. configure the plugin at the option page
1. you are done

== Frequently Asked Questions ==
= How can i add new icons? =
At the moment there is no way to use your own icons. But I plan to make this for future versions.
